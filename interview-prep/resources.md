# Interview Prep Resources

> Last updated: 2026-04-18. Focused on AI/ML eng roles (Anthropic, OpenAI, ByteDance-level).

## AI/ML-Specific (2025–2026)

### ai-interview-codex
- **Repo**: https://github.com/girijesh-ai/ai-interview-codex
- **Covers**: LLM fundamentals, RAG system design, agentic AI patterns, traditional ML
- **Standout**: Production-ready code examples; inference pipeline design with latency constraints; RAG end-to-end system design
- **Algo gap-filler**: Attention mechanism + gradient descent from scratch (live coding, not just theory)
- **Best for**: Anthropic / OpenAI / ByteDance ML rounds

### ai-engineering-field-guide
- **Repo**: https://github.com/alexeygrigorev/ai-engineering-field-guide
- **Covers**: Take-home challenge breakdowns, hiring practice data Q4 2025/Q1 2026 across 51 companies
- **Standout**: Company-specific interview patterns — more actionable than Glassdoor
- **Best for**: Pre-interview company research, understanding what a specific team actually tests
