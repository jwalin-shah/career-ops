# Issue Tracker

**Platform:** GitHub Issues

**Repository:** `jwalinshah/career-ops`

**CLI:** `gh issue` (create, list, view, close)

**PRs as request surface:** No. External PRs are not triaged as issues. Collaborator PRs are handled directly.

**CI/CD:** GitHub Actions run `test-all.mjs` (63+ checks) on every PR. Branch protection on `main` requires status checks to pass.
