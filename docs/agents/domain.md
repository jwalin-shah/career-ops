# Domain Docs

**Layout:** Single-context

**Files:**
- `CLAUDE.md` at repo root — project contract and instructions
- `AGENTS.md` at repo root — agent-specific overrides
- `DATA_CONTRACT.md` — user layer vs system layer boundaries
- `docs/` — architecture, setup, and customization docs

**Career-ops-specific domain:**
- `modes/` — Claude Code skill modes (evaluation, pipeline, scan, etc.)
- `config/` — user profile, portal config, model settings
- `data/` — application tracker, pipeline, follow-ups
- `templates/` — CV templates (HTML, LaTeX), states, portals
- `reports/` — evaluation reports (A-G scoring)
- `batch/` — batch processing with parallel workers
- `scripts/` — `*.mjs` utilities: scan, liveness, PDF generation, merge, verify
